#  Healthcare Data Analysis Project

##  Project Overview

This project focuses on analyzing healthcare data to uncover key insights related to patient demographics, hospital performance, billing trends, and medical conditions.

The analysis was performed using SQL to transform raw data into meaningful business insights. Additionally, an interactive dashboard was built using Power BI to visualize key metrics and trends, enabling better decision-making and data storytelling.

The project demonstrates how data analysis and visualization can be combined to improve hospital efficiency, reduce costs, and enhance patient care.

---

## Objectives

* Analyze patient demographics (age, gender, blood group)
* Evaluate hospital and doctor performance
* Identify high-cost treatments and revenue drivers
* Track admission trends over time
* Analyze insurance provider efficiency
* Measure patient stay duration and readmission patterns

---

##  Tools & Technologies

* SQL (PostgreSQL)
* Data Analysis Techniques
* Power BI (for dashboard visualization)

---

##  Dataset Description

The dataset contains the following fields:

* Patient Information: name, age, gender, blood_type
* Medical Details: medical_condition, medication, test_results
* Hospital Details: hospital, doctor, room_number
* Admission Details: admission_type, date_of_admission, discharge_date
* Financial Data: billing_amount, insurance_provider

---

##  Data Cleaning

Performed the following data cleaning steps:

* Checked for NULL values in critical columns
* Identified duplicate patient records
* Validated data ranges (age, billing amount)
* Extracted unique medical conditions and categories
